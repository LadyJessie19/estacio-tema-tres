# 🃏 Super Trunfo em C - Tema 2 🚀

Bem-vinda ao desafio de lógica da **TechNova**! Este projeto simula o clássico jogo Super Trunfo, onde comparamos atributos de cidades para determinar a vencedora.

## 🛠️ O que o programa faz?

- **Cadastro de Cartas**: Recebe dados como População, Área e PIB.

- **Cálculos Automáticos**: Gera a **Densidade Populacional** e o **PIB per capita**.

- **Menus Interativos**: Você escolhe quais atributos quer comparar usando um menu `switch`.

- **Lógica Inteligente**: Ganha quem tem o maior valor, exceto na **Densidade**, onde o menor valor vence!

---

## 🏗️ Como rodar o projeto?

Siga estes passos simples no seu terminal:

1. **Compilar o código**:

```bash
gcc super-trunfo-c.c -o super-trunfo-c

```

2. **Executar o jogo**:

```bash
./super-trunfo-c

```

---

## 🎮 Instruções de Uso

- 📝 **Passo 1**: Insira as informações da primeira cidade (ex: São Paulo).

- 📝 **Passo 2**: Insira as informações da segunda cidade (ex: Rio de Janeiro).

- 🔢 **Passo 3**: Escolha dois atributos diferentes no menu para a batalha.

- 🏆 **Passo 4**: O programa somará os pontos e anunciará a carta vencedora!
